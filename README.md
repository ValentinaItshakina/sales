## Описание классов и их методов
##№ Класс Product

###№ Описание
Класс представляет товар на складе с уникальным идентификатором, названием, ценой и количеством.

#### Поля
| Модификатор | Тип | Название | Описание |
|-------------|-----|----------|----------|
| private | std::string | productId | Уникальный идентификатор товара |
| private | std::string | name | Название товара |
| private | double | price | Цена товара |
| private | int | quantityInStock | Текущее количество на складе |

#### Конструкторы
| Сигнатура | Описание |
|-----------|----------|
| `Product()` | Конструктор по умолчанию |
| `Product(const std::string& productId, const std::string& name, double price, int quantityInStock)` | Конструктор с параметрами для создания товара |

#### Методы доступа (геттеры/сеттеры)
| Модификатор | Возвращаемый тип | Сигнатура | Описание |
|-------------|------------------|-----------|----------|
| public | std::string | `getProductId() const` | Возвращает ID товара |
| public | std::string | `getName() const` | Возвращает название товара |
| public | double | `getPrice() const` | Возвращает цену товара |
| public | int | `getQuantityInStock() const` | Возвращает количество на складе |
| public | void | `setName(const std::string& name)` | Устанавливает название товара |
| public | void | `setPrice(double price)` | Устанавливает цену товара |
| public | void | `setQuantityInStock(int quantity)` | Устанавливает количество товара |

#### Основные методы
| Модификатор | Возвращаемый тип | Сигнатура | Описание |
|-------------|------------------|-----------|----------|
| public | void | `restock(int quantity)` | Увеличивает количество товара на указанное значение |
| public | bool | `sell(int quantity)` | Уменьшает количество при продаже. Возвращает `true` если достаточно товара и продажа успешна |
| public | bool | `isAvailable() const` | Проверяет наличие товара (`quantityInStock > 0`) |

#### Статические методы
| Модификатор | Возвращаемый тип | Сигнатура | Описание |
|-------------|------------------|-----------|----------|
| public static | bool | `isValidPrice(double price)` | Проверяет корректность цены (вероятно, > 0) |

#### Дополнительные методы
| Модификатор | Возвращаемый тип | Сигнатура | Описание |
|-------------|------------------|-----------|----------|
| public | void | `print() const` | Выводит информацию о товаре в консоль |

#### Перегруженные операторы
| Оператор | Сигнатура | Описание |
|----------|-----------|----------|
| `==` | `bool operator==(const Product& other) const` | Сравнивает товары (вероятно, по ID) |
| `<<` | `friend std::ostream& operator<<(std::ostream& os, const Product& product)` | Выводит товар в поток |
